CPU vs RAM для сетевых задач и VPN: куда утекает узкое место

Для VPN-узла CPU тратится на криптографию и обработку пакетов на высоких скоростях. RAM важна для таблиц соединений (conntrack), буферов сокетов, кэшей маршрутизации и самого userspace-процесса. Нет универсального «берите только RAM».

Лучше избегать: латентность взлетает.

Что такое RSS?

Receive Side Scaling — распределение прерываний по ядрам.

Как RAM влияет на TLS?

Косвенно через буферы и количество одновременных рукопожатий.
